# Substrato 6160 — GECC Full Simulation

This repository contains the simulation of the Genomic Error-Correction Code (GECC).

## Execution
Run `python3 substrato_6160_gecc.py` to compile the genome, run the simulation, anchor the ArtBlock, and generate the PDF report.

The simulation proves the concept that "junk DNA" serves as parity bits for cosmic error correction, maintaining a high fidelity (>99%) across replication cycles.
