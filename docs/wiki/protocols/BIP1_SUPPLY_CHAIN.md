# BIP-1: Cadeia de Suprimentos (Tier 1)

Para garantir a integridade do Bio-Implante de Pulso, o Complexo Integrado CORVO (CIC) homologou os seguintes fornecedores para materiais de grau médico e usinagem de precisão.

## 1. Titânio de Grau Médico (Grade 23 ELI)
O chassi do BIP-1 exige Titânio Grau 23 (Ti-6Al-4V ELI), otimizado para biocompatibilidade e resistência à fratura.

*   **Carpenter Technology (EUA):** Fornecimento de barras de titânio certificadas ASTM F136.
*   **Zapp Precision Metals (Alemanha):** Especialista em fios e barras de precisão para micro-usinagem.
*   **Sandvik Coromant (Suécia):** Ferramental para usinagem criogênica de alta precisão.

## 2. Janela de Safira e Zircônia
*   **GT Advanced Technologies:** Safira óptica para interface fotônica.
*   **Kyocera:** Cerâmicas de Zircônia Negra para o isolamento térmico e estético do implante.

## 3. Especificações de Fabricação
*   **Acabamento:** Anodização Tipo II (Cinza/Preto).
*   **Certificação:** ISO 13485 (Medical Devices).
