# Awesome AI Apps Source Repository 🜏

A comprehensive collection of 80+ practical AI examples, tutorials, and recipes for building powerful LLM-powered applications. This repository serves as a foundational knowledge source for experiment adoption in the Arkhe(n) Proof of Intelligence (PoI) economy.

## 🧩 Categories of Intelligence

### 1. Starter Agents
Quick-start agents for learning AI frameworks:
- **Agno HackerNews Analysis:** Trend analysis on HackerNews.
- **OpenAI SDK Starter:** Email helpers and haiku writers.
- **LlamaIndex Task Manager:** Powered task assistant.
- **CrewAI Research Team:** Multi-agent research examples.

### 2. Simple Agents
Straightforward use-cases for everyday AI:
- **Finance Agent:** Real-time stock and market data tracking.
- **HITL Agent:** Human-in-the-loop actions for safe execution.
- **Newsletter Generator:** AI-powered builder with Firecrawl.
- **Agent Discovery:** Find/compare agents across NANDA, MCP, Virtuals.

### 3. Voice Agents
Real-time assistants and speech pipelines:
- **LiveKit + Gemini Realtime:** Low-latency voice rooms.
- **Pipecat + Sarvam:** Voice pipelines with STT/TTS.

### 4. MCP Agents
Model Context Protocol for external tool integration:
- **Doc-MCP:** Semantic RAG documentation.
- **GitHub MCP:** Repository insights and analysis.
- **Docker E2B MCP:** Secure sandboxed environments.

### 5. Memory Agents
Advanced context retention and personalization:
- **arXiv Researcher with Memori:** GPT/GibsonAI Memori integration.
- **Brand Reputation Monitor:** Sentiment tracking and news analysis.
- **YouTube Trend Agent:** Analysis using Memori, Agno, and Exa.

### 6. RAG Applications
Retrieval-augmented generation for document understanding:
- **Agentic RAG:** Agno & GPT-5 implementation.
- **Nvidia Nemotron OCR:** Document/Image parsing using 12b model.
- **WFGY LLM Debugger:** 16-mode map based debugger.

### 7. Advanced Agents
Complex multi-agent pipelines:
- **Nebius AutoResearch:** Taxi analytics pipeline optimizer.
- **Deep Researcher:** Multi-stage research with ScrapeGraph AI.
- **AI Hedgefund:** Comprehensive financial analysis workflows.
