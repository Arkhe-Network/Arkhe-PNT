# Global OSINT Toolkit (Awesome OSINT) 🌐

## Overview
This document synthesizes the global OSINT toolkit from the [Awesome OSINT For Everything](https://github.com/Astrosp/Awesome-OSINT-For-Everything) repository (see [[../../sources/awesome-osint|Raw Source Record]]). These tools provide a global reach for mapping, validating, and protecting the **Z-Domain** (physical reality) and ensuring its alignment with the **C-Domain**.

## Strategic Relevance
To maintain a high Kuramoto order parameter ($R$) across global nodes, the Cathedral utilizes these tools for:
1. **Global Node Validation**: Vetting international operators and organizations.
2. **Infrastructure Resilience**: Monitoring global transport, maritime, and aviation for logistics and phase-shielding.
3. **AI & LLM Security**: Protecting the Cathedral's cognitive layers from entrainment or prompt injection attacks.
4. **Threat Intelligence**: Real-time monitoring of global conflicts and cyber threats.

## Key Resource Categories

### 1. Unified Search & People Discovery (Node Vetting)
- **SynapsInt**: Comprehensive search for domains, IPs, and ASNs.
- **WhatsMyName**: Enumerate usernames across multiple global platforms.
- **Epieos**: Reverse email and phone lookup for identity resolution.
- **Sherlock**: Social media account hunting by username.

### 2. Threat Intelligence & Real-Time Monitoring
- **Pulsedive**: Community-driven threat intelligence with risk scoring.
- **GreyNoise**: Filtering "internet noise" to identify actual scanner activity.
- **Liveuamap**: Real-time conflict tracking for global "hot zones."
- **Cyberthreat Real-Time Map (Kaspersky/Checkpoint)**: Visualizing global attack vectors.

### 3. AI & LLM Security (Cognitive Protection)
- **Garak**: LLM vulnerability scanner for probing language models.
- **PromptFoo**: Evaluating LLM outputs for security and quality.
- **MITRE ATLAS**: Knowledge base of adversarial tactics against ML systems.
- **OWASP Top 10 for LLM**: Identifying critical security risks in AI applications.

### 4. Infrastructure & Logistics (Beacon Expansion)
- **ADS-B Exchange**: Uncensored flight tracking for aerial relay nodes.
- **Marine Traffic / VesselFinder**: Real-time maritime tracking for underwater fiber relays.
- **Shodan / Censys**: Mapping the global attack surface of internet-connected devices.
- **WiGLE**: Global wireless network mapping.

### 5. Blockchain & Crypto Investigation (Financial Integrity)
- **Chainabuse**: Reporting and searching crypto scam addresses.
- **OXT.me**: Bitcoin blockchain explorer with advanced analysis tools.
- **Arkham Intelligence**: Entity tracking and crypto intelligence.
